Sobre Nosotros

Alcance

Esta página web sobre información metereológica surge del desafío "¿Lloverá en mi desfile", donde se plantea la búsqueda de una solución a cuándo se quiere saber las condiciones climáticas de un lugar, fecha y hora en específico, pero con la diferencia que supera a los 15 días que se suelen encontrar en otros sistemas.

Características Principales:

  • Datos meteorológicos en tiempo real
  • Análisis deportivo basado en probabilidades por cada deportes
  • Predicciones hasta 1 año
  • Exportación de datos JSON/CSV

Fuentes de Datos

Para realizar este proyecto utilizamos información de múltiples fuentes a las que puede acceder haciendo click sobre la que desee:

NASA GES DISC

Datos satelitales históricos y reanálisis atmosférico

OpenMeteo API

Pronósticos precisos hasta 5 días

Data Rods for Hidrology

Datos hidrológicos históricos
